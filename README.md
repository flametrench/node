# Flametrench Node SDK

Node SDK for [Flametrench](https://github.com/flametrench/spec). Monorepo of `@flametrench/*` packages.

## Packages

- [`@flametrench/ids`](./packages/ids) — Prefixed wire-format IDs for Flametrench.

More packages will land as the v0.1 specification stabilizes.

## Development

```bash
pnpm install
pnpm test
pnpm build
```
